Getting Started

Get up and running with the Golden Codex API in under 5 minutes.

1

Get Your API Key

  1. Sign up at golden-codex.com
  2. Navigate to Dashboard → API Keys
  3. Click Create New Key
  4. Copy the key immediately - it won't be shown again!

Your API key looks like: gcx_live_Abc123XyzDef456...

Security Note Never commit API keys to version control. Use environment variables instead.
2

Install an SDK

Node.js

npm install @golden-codex/sdk

Python

pip install golden-codex
3

Process Your First Image

Node.js

import { GoldenCodex } from '@golden-codex/sdk';

const gcx = new GoldenCodex({
  apiKey: process.env.GOLDEN_CODEX_API_KEY
});

// Create a job
const job = await gcx.jobs.create({
  imageUrl: 'https://example.com/artwork.jpg',
  operations: ['nova', 'flux', 'atlas']
});

console.log(`Job created: ${job.jobId}`);

// Wait for completion
const result = await gcx.jobs.wait(job.jobId);

console.log('AI Title:', result.results.goldenCodex.title);
console.log('Final URL:', result.results.urls.final);

Python

from golden_codex import GoldenCodex
import os

gcx = GoldenCodex(api_key=os.environ["GOLDEN_CODEX_API_KEY"])

# Create a job
job = gcx.jobs.create(
    image_url="https://example.com/artwork.jpg",
    operations=["nova", "flux", "atlas"]
)

print(f"Job created: {job['job_id']}")

# Wait for completion
result = gcx.jobs.wait(job["job_id"])

print("AI Title:", result["results"]["golden_codex"]["title"])
print("Final URL:", result["results"]["urls"]["final"])

Understanding Operations

Operation What It Does Cost
nova AI analyzes image and generates 50+ metadata fields 1-2 AET
flux Upscales image 2-4x using ESRGAN 1-2 AET
atlas Embeds metadata into image file (XMP/EXIF) 1 AET

You can run any combination of operations:

// Just metadata, no upscaling
await gcx.jobs.create({
  imageUrl: '...',
  operations: ['nova', 'atlas']
});

// Just upscaling, no metadata
await gcx.jobs.create({
  imageUrl: '...',
  operations: ['flux']
});
